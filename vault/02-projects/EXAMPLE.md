# Example Project

Template for a project note. Copy this to `<project>.md` — which stays **local and
gitignored** — and fill it in. Real project notes are never committed to the public repo;
a memory vault is private by design.

- **What:** one-line description of the project.
- **Location:** local path (kept private).
- **Status:** current state in a line or two.
- **Gotchas:** environment quirks worth remembering.
