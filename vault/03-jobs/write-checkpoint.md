# Job: write-checkpoint

**Purpose:** capture a compact checkpoint so work can resume cleanly in a fresh session.

**Prime on:** this note + the active project note + today's `01-daily/` log.

## Steps

1. Summarize the current state in **≤12 bullets**.
2. State explicitly: branch, goal, protected files, validation commands, commit boundary.
3. Write it into the project note's **State**, and append it to today's daily log.
4. Do not paste long logs — summarize only the failing lines and the command that produced them.

## Output

The ≤12-bullet checkpoint plus the single exact next decision. Nothing more.
